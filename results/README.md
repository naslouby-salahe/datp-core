# results/

Curated, lightweight, shareable derivations only: result tables, figures, and
the claim-evidence map. Raw scores, checkpoints, and raw tables/figures live
under `outputs/` and `checkpoints/`, not here.

Populated by result curation (P7-T06) from `outputs/`, never written to
directly by an experiment stage. See
[docs/protocol/artifact_contracts.md](../docs/protocol/artifact_contracts.md)
§2 for the curated-vs-raw artifact distinction.
