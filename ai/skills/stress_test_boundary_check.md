# Stress Test Boundary Check

## Purpose
Keep stress tests outside the core B1-B4 causal ladder.

## When to apply
Apply when FedProx, model personalization, external validation, comparators, or boundary experiments are discussed or changed.

## Blocking rules
Block wording or code structure that makes stress tests replace the causal ladder or change the trained-model premise.

## Pass criteria
Stress tests are supportive, boundary, or exploratory and are not used to redefine confirmatory evidence.

## Fail criteria
Stress tests are promoted to primary causal evidence or used to broaden DATP into generic FL-IDS claims.
