# README Makefile Sync Check

## Purpose
Keep documented commands and automation accurate.

## When to apply
Apply after command, config, output, workflow, or Makefile behavior changes.

## Blocking rules
Block stale README commands, stale Makefile targets, old config names, old output paths, old CLI flags, and docs that preserve legacy behavior.

## Pass criteria
README and Makefile match current commands, config names, output paths, and workflow.

## Fail criteria
Docs instruct stale usage or keep old names alive.
