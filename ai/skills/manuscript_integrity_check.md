# Manuscript Integrity Check

## Purpose
Keep manuscript and documentation claims accurate.

## When to apply
Apply to abstracts, introductions, related work, methods, results, captions, discussion, conclusion, README prose, and reviewer responses.

## Blocking rules
Block unsupported claims, robust claims without tests, privacy-preserving claims without proof, deployment-ready claims without profiling, first claims without verification, overgeneralized wording, hype, and AI-looking filler.

## Pass criteria
Claims are evidence-backed, bounded, classified, and written without hype.

## Fail criteria
Scientific meaning is changed, evidence is overstated, or future work is written as completed work.
