### B2. Journal-planning boundary

No Phase 5 activity performs real journal scientific execution. Do not train, score, threshold-sweep, run a reduced/debug cell, write scientific outputs, infer seed order, or use observed outcomes to change the matrix, feasibility decision, reuse decision, resource plan, or frozen configuration.

A journal plan is fail-closed: every cell has resolved identity and feasibility; compatible reuse is proven from complete lineage, never names, paths, list position, or seed position; an identity-bearing change requires new computation; a threshold-only change does not retrain or rescore. A changed scientific configuration creates a new campaign identity. Preserve the distinction among resume, recovery, transient retry, technical-invalidity correction, affected-stage recomputation, a new execution attempt, and a new campaign.
